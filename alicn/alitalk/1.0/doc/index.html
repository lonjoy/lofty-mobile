Alitalk : 阿里旺旺
王涛

示例

介绍

阿里旺旺组件

使用方法

 
define(['alicn/alitalk/1.0'],function( Alitalk ){
  Alitalk.init({
    trigger:$('a[data-alitalk]', '#demo1')
  });
})

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
cls Json Object { base: 'alitalk', on: 'alitalk-on', off: 'alitalk-off', mb: 'alitalk-mb'} 不同状态下旺旺图标的class name
attr String 'alitalk' 组件信息所在的标签（默认会先判断data-前缀）
siteID String 'cnalichn' 旺旺所在的站点
remote Boolean true 是否通过接口获取用户在线状态
onRemote Function Function 每个DOM在获取到用户状态信息之后，调用此方法。可以重写此方法来适应不同的应用需求
onSuccess Function NULL 所有DOM获取到用户状态信息之后，调用此方法。该方法与onRemote的区别是，onRemote会执行多次
prop Function Function 传递给桌面客户端的额外信息
getAlitalk Function Function 未安装或未检测到安装旺旺后调用此方法

公共方法

方法名称 入参说明 返回值 作用说明
init (options:Object) 组件的配置参数 / Alitalk初始化函数
login (id:String) 用户的loginId / 调用插件的登录功能，触发桌面客户端的登录窗口
getVersion () / Number 返回当前旺旺的 version 版本
installed () / true | false 判断旺旺插件是否安装
