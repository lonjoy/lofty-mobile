Resource Slot : 在主客H5页面投放的资源位广告
袁力皓

该组件用在阿里主客上的插件显示投放的广告，广告类型包括轮播型、纯文字性以及单图片型（包含图文型）。

注：因为该组件底层依赖于Wing，所以下方示例中的demo的数据来源都是mock出来的。

示例

使用方法

 
    define(['lofty/ui/resourceslot/1.0/resourceslot'], function(resSlot){
        resSlot.create({
            container: '.wrapper',
            scene: {
              place: '1',
              keyword: 'whatever'
            },
            sys: {
              modifyTime: '20140324',
              city: '',
              appVer: '',
              osVer: '',
              userId: '',
              ttid: '',
              imei: '',
              imsi: ''
            }
        });
    })

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
container(必填) string 放置广告内容的容器selector
scene object / 包含place/keyword等几个属性
place string / 广告场景，由运营方配置，使用者可直接联系运营方获取各种广告类型相对应的place值
keyword string / 关键词
sys object / 包含modifyTime/city/appVer/osVer/userId/ttid/imei/imsi等属性
modifyTime string / 修改时间，格式为yyyyMMdd
city string / 当前城市
appVer string / 应用版本号
osVer string / 系统版本号
userId string / 用户id
ttid string / 渠道号
imei string / 手机imei
imsi string / 手机imsi
