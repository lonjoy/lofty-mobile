switch : 支持手势切换的轮播组件、切换组件
沈剑平

示例

介绍

支持手势切换的轮播组件、切换组件

使用方法

 
    define(['fui/swipe/1.0'],function(Swipe){
        var swipe  = new Swipe({
            tpl : "#swipe"
        });
    })

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
tpl String / swipe组件内容的class或id
startSlide number 0 初始的slider 帧数
speed number(毫秒) 400 切换速度
auto number(毫秒) 3000 切换的时间间隔(设置0表示必须手动切换)
continuous Boolen true 是否连续
disableScroll Boolen false 切换的时候，阻止touch事件传播
stopPropagation Boolen false 阻止事件传播
callback Function null 切换的callback （ function(index, elem) {} ）
pane String .swipe-pane 默认的slider 内容选择器
nav Strign .swipe-nav li 默认的slider trigger 选择器
active String active 默认的slider trigger 选择器高亮Class
imgLazySrc String swipe-lazy-src slider图片懒加载，默认的实际图片地址attr

公共方法

方法名称 入参说明 返回值 作用说明
slide(idx) (idx:Number) / 切换到某帧
prev() / / 切换到上一帧
next() / / 切换到下一帧
getPos / Number 返回当前的帧数
