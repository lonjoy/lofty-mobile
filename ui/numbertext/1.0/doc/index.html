示例

介绍

数字加减的DPL，JS逻辑需要业务自己实现

使用方法

 
直接拷贝demo中的DOM结构，并引入默认的CSS： 
<link rel="stylesheet" href="http://style.c.aliimg.com/m/lofty/ui/numbertext/1.0/numbertext.css" />



如果需要自定义颜色，宽度，高度等 ，可以新建 less 文件，并自定义部分变量：

@import url(http://style.c.aliimg.com/m/lofty/ui/numbertext/1.0/numbertext.less);

@numberHeight:60px;  //数字文本域的高度
@numberWidth:114px;  //数字文本域的宽度
@numberBgColor:#fff; //数字文本域的背景色
@numberPadding:0 3px; //数字文本域的padding值
@optionBgColor:#fafafa; //操作按钮背景色
@optionActiveBgColor:#eaeaea; //点击操作按钮时的背景色
@borderColor:#d9d9d9; //border的颜色
@optionTxtColor:#999; //"+"和"-"的字体颜色
@optionTxtActiveColor:#999; //"+"和"-"的字体颜色
@borderRadius:3px;    //圆角的大小
